# Linux Device Driver
As part of this course, we explore how to go about writing a Linux Device Driver module.




## What would you learn during this project?
1. What is Linux Kernel and the role of device drivers
2. Basic components of a Linux Device Driver.
3. Compiling the device driver code to a loadable kernel module.
4. Understanding the models of interactions between User space and Kernel Space
5. Interacting with the Kernel Module using a user-space application
6. Various Utilities involved in developing, loading and unloading the kernel module
